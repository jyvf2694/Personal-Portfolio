Stella Artois

This project is developed using Drupal as a content management system, providing a flexible and powerful platform for managing websites. The hosting infrastructure is supported by Acquia Cloud Site Factory, allowing efficient management of multiple sites within a Drupal Multisite environment.

For data analysis and user tracking, tools such as TikTok Pixel, Google Analytics, and Facebook Pixel are integrated. Security is reinforced with Imperva and HSTS, protecting the site from threats and ensuring a secure browsing experience for users.

Regarding the interface, the project uses Tailwind CSS and Bootstrap for design, while JavaScript libraries such as AOS, jQuery, and OWL Carousel provide a dynamic and engaging experience. The network is optimized with Varnish as a caching tool, and performance is enhanced using CDNs like Cloudflare and jsDelivr.
